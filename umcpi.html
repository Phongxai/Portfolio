Project

UMCPI

  • Web Design @ ADG
  • Front End Development @ ADG

UMCPI, a non profit that offers training and consulting to public safety agencies, needed a website redesign because their old site was outdated.

The new website was built with mobile first in mind. Using CSS media queries, the website is fully responsive and looks great both on desktop and mobile. The website was built with wordpress for ease of updates without a developer.
